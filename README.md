To run Application :-
1). git clone {paste the link of github project}
2).Open this project in vscode.
**For Client side ---**
3). In terminal run **cd client** command
4). run **npm i** command
5). run **npm start** command.

**For server side ---**
6). In terminal run **cd server** command
7). run **npm i** command
8). run **npm start** command.
